## ✨ Core Systems (Features)

* **🖥 Arch-Terminal UI**: A bespoke "Neofetch" inspired component displaying system data and engineering stats.
* **📊 Bento Grid Architecture**: Sophisticated layout separating "Achievements," "Side Quests," and "Company Work."
* **🏆 The Growth Timeline**: A visual record of evolution from 2023 to 2025, including Gold Awards and AI Olympiad Finalist status.
* **🌑 Elite Dark Mode**: A custom-engineered UI using Tailwind CSS, Framer Motion, and grain-textured glassmorphism.
* **SEO Optimized**: Fully integrated with Next.js Metadata API, Google Search Console, and automated Sitemaps.

<br>

## 🛠 Engineering Stack

* **Framework**: Next.js 14/15 (App Router)
* **Styling**: Tailwind CSS (Custom Design System)
* **Animation**: Framer Motion (Reveal-on-scroll)
* **Icons**: Lucide React
* **Analytics**: Google Search Console
